🎉 **`Project` :** 가상화폐 상승장/하락장 알리미<br>
🎉 **`팀원` :** 김민주, 김여름, 한지희

## 개선사항
✨ 마우스로 셀을 클릭하면 해당 셀의 헤더를 굵게 표시<br>
✨ 상승장은 초록색으로 표시, 하락장은 빨간색으로 표시<br>
✨ 한눈에 보기 쉽도록 창 크기에 맞춰 자동으로 크기가 조절되도록 설정
